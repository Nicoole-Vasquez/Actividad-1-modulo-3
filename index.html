Investiga en diferentes sitios web y describe cada tema:

  1. Definición de CSS:

    2. Es un lenguaje que se utiliza para definir el diseño y la apariencia de un documento HTML, permitiendo controlar estilos como colores, fuentes, tamaños y la disposición de los elementos en pantalla.

  2. Funcionamiento básico de CSS:

    3. CSS funciona aplicando reglas que consisten en un selector para identificar elementos HTML y un conjunto de propiedades con valores que especifican cómo deben lucir esos elementos.

  3. Incluir CSS en un archivo HTML:

    4. CSS puede integrarse a un archivo HTML de forma inline, escribiendo estilos directamente en elementos, interna, dentro de etiquetas style, o externa, enlazando un archivo separado con link.

  4. Definición de una regla CSS:

    5. Una regla CSS es una instrucción que aplica estilos a elementos HTML específicos mediante un selector y un bloque de declaraciones con propiedades y valores.

  5. Elemento de una regla CSS:
    • Regla:

      • Una regla CSS es una instrucción que define cómo se deben mostrar ciertos elementos HTML. Consiste en un selector y un bloque de declaración que especifican las propiedades y sus valores.

    • Selector:

      • El selector es la parte de la regla CSS que indica a qué elementos HTML se les aplicarán los estilos. Puede ser un tipo de etiqueta, una clase, un ID o cualquier combinación de estos.

    • Bloque de declaración:

      • El bloque de declaración es la parte de la regla CSS que contiene las propiedades y sus valores, que definen el estilo de los elementos seleccionados. Se encierra entre llaves {}.

    • Propiedad:

      • La propiedad es el aspecto visual que se quiere modificar de un elemento HTML, como el color, el tamaño de la fuente o el margen. Se define dentro del bloque de declaración.

    • Valor:

      • El valor es la configuración específica que se asigna a una propiedad. Puede ser un color, una medida, una fuente, entre otros, dependiendo de la propiedad que se esté modificando.

  6. Medio CSS:

    7. Los medios CSS (o consultas de medios) permiten aplicar estilos diferentes a una página web según características específicas del dispositivo o entorno, como el tamaño de la pantalla, la orientación o la resolución. Se utilizan principalmente para crear diseños adaptativos o responsivos.

  7. Comentario CSS:

    8. Los comentarios en CSS se utilizan para agregar notas o descripciones en el código sin afectar la ejecución del mismo. Se escriben entre /* y */ y son útiles para documentar el código o desactivar temporalmente ciertas reglas.

  8. Sintaxis de la definición de las propiedades CSS:

    9. La sintaxis para definir propiedades en CSS consiste en especificar la propiedad seguida de un valor, separados por dos puntos, y terminar con un punto y coma. La propiedad se escribe en minúsculas, y el valor depende de la propiedad (por ejemplo, un color o un tamaño).
